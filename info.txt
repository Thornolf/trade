Calculer le coef directeur tous les N paliers pour identifier l'evolution de la courbe

Si l'indice n - 1 est supérieur à l'indice n - 2 on achete, sinon on vend

Pseudo-Code :


*_*_*_*_*_* MAIN  *_*_*_*_*_*_*

IndiceActuel = 0;
TableauDesIndices[]
IndexTableau = 0
Palier = Ratio de nombre de palier en fonction du nombre de NombreDeJour

while IndiceActuel < NombreDeJour {
	if (IndiceActuel % Palier == 0) {
		TableauDesIndices[IndexTableau] = IndiceActuel;
		if (TableauDesIndices.size > 2) {
			DoWeHaveToSellOrBuyOrWait(); // Fonction qui nous dit si l on doit vendre ou acheter ou attendre
			}

		}
		IndexTableau++;
	}
}

*_*_*_*_*_*_* DoWeHaveToSellOrBuyOrWait *_*_*_*_*_*_*

Effectué un calcul qui déterminera si l'on doit acheter vendre ou attendre,
Sachant que l'on esst taxé d'un % quand l'on vend ou on achete, il faut donc ne pas vendre ni
ne pas acheter en permanence mais attendre parfois.

On doit retenir quel jour et à quel indice l'on a acheter ou vendu, ainsi avant de validé
l'achat on vérifie les achats antérieurs ou les ventes antérieurs et on regarde le
% de profit ou de perte et on en conclut si oui ou non on doit vendre

Si c'est le dernier jour, on vent toutes nos actions.
